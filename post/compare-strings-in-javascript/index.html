发表于 2023-02-26 分类于 Mastering JS

原文：Compare Strings in JavaScript

JavaScript的比较运算符很容易比较字符串，另外还有localeCompare()可以用于排序。

JavaScript中，两个字符串相等代表什么？一个字符串大于另一个又代表什么？

JavaScript的三等运算符===，在两个字符串完全相同时返回true，否则返回false：

<和>运算符按词法顺序比较字符串。词法顺序基本上等同于“字典顺序”。简单的说，如果按ASCII表顺序，a的第一个字符在b的第一个字符之前，那么a < b；如果a和b的第一个字符相等，而a.slice(1) < b.slice(1)，那么也是a < b。

字符串数组排序

JavaScript的Array#sort()函数按词法顺序自动排序。

lodash的sortBy()函数对字符串数组也默认按词法顺序排序。

一个对象数组，如何按字符串属性排序？Array#sort()接收一个回调函数作为参数，回调接收2个值a和b，返回值如下：

  • 0 如果a和b相等
  • 正数 如果a > b
  • 负数 如果a < b
