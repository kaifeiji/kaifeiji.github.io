JavaScript中的Object.assign()

发表于 2023-01-26 分类于 Mastering JS

原文：Object.assign() in JavaScript

Object.assign()函数将一个对象的属性赋给另一个对象，可以用于浅拷贝对象或一次性赋值多个属性。

JavaScript的Object.assign()函数从一个或多个来源对象复制属性到目标对象，它的返回值是目标对象。

虽然展开运算符...做浅拷贝一般比Object.assign()更快，但Object.assign()也经常用于浅拷贝对象。例如在Redux的recucer中就经常需要进行浅拷贝。

多个源对象

Object.assign()可以传入多个源对象，如果多个源对象中有相同的属性，以最后一个的属性为准。
