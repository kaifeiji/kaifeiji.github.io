JavaScript的Object.keys()函数概览

发表于 2023-01-15 分类于 Mastering JS

原文：An Overview of the Object.keys() Function in JavaScript

介绍如何使用JavaScript的Object.keys()。

Object.keys()函数返回一个对象的属性名的数组。例如一个POJO对象obj，调用Object.keys()会返回对象所有的key。

key的顺序

属性名是按照属性设置的顺序排列的。如果delete一个属性，然后再次设置该属性，它会位于数组的末尾。

ECMAScript规范称之为“属性创建顺序”。Object.keys()函数在所有ES2015兼容的环境中，可以保证返回的key按属性创建顺序。但有一个关键性的例外：数值型key。

任何属性名是介于0和2^32 - 1（包含）之间的整数，会位于所有非整数key之前，而且这些属性会按数值排序。

ES6的类

Object.keys()函数只返回所谓的“自有属性”。因此Object.keys()不会返回类方法或类属性。

Symbol

Object.keys()的属性不包括Symbol类型的属性名。需要用Object.getOwnPropertySymbols()来获取Symbol属性。
