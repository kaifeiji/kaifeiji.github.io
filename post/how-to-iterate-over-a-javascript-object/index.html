如何遍历JavaScript的对象

发表于 2023-02-23 分类于 Mastering JS

原文：How to Iterate Over a JavaScript Object

有许多种方法可以遍历JavaScript对象的所有key和value。本文介绍3种方法和它们的优缺点。

如何遍历key/value对，并打印“name: Luke Skywalker”、“title: Jedi Knight”和“age: 23”？现代JavaScript种有许多方法实现，本文介绍了3种不同的方法：

Object.entries()

Object.entries()函数返回一个数组，其中包含对象的key/value对。使用Object.entries()和for/of循环，可以打印key/value对。

Object.entries(obj)返回的数组，其中每个都包含2个元素，第一个是key，第二个是value。这种模式在JavaScript中很常见，例如Map的constructor和Object.fromEntries()。

如果用Array.from()将Object.entries()返回的迭代器转换为数组，数组中包含的就是对象的key/value对。

Object.keys()

Object.keys()函数返回对象key的数组，而不是key/value对。可以用方括号[]获取对象的value。

有了Object.entries()为什么还要有Object.keys()？因为entry中的值是无法改变对象中的值的。

for/in

Object.keys()和Object.entries()只会循环对象自身的属性。对于POJO而言，区别不大，但在遇到继承的情况，可能会有一些问题。for/in循环可以遍历对象的所有key，包括继承的key。

一般来说，应该使用Object.keys()或Object.entries()遍历POJO，避免意外取到继承属性。但如果确实需要遍历继承属性，就需要用for/in了。
